Java Review

2017-07-26
  1. 把衍生类型当做它的基本类型处理的过程叫做“Upcasting”（上溯造型），这是避免去调查准确类型的好办法
  2. 用abstract关键字描述一个尚未实现的方法。继承了一个类后，那个方法就必须实现，否则继承的类也会变成”抽象”类。若试图创建抽象类的一个对象，编译器就会阻止
  3. 上溯造型是安全的，下溯造型很难保证安全
  4. 类成员默认值

主类型 默认值
Boolean false
Char ‘\u0000’(null)
byte (byte)0
short (short)0
int 0
long 0L
float 0.0f
double 0.0d

如果是局部变量，并非一个类的字段。变量将会得到随机值，而不会自动初始化成零

  1. 软件包以小写字母为标准
  2. 所有Javadoc命令都只能出现于”/*”注释中，注释结束于一个”\/“
