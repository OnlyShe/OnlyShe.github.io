Jsp学习记录

2017-07-25

  1. response.encodeURL()在不支持Session的浏览器自动重写URL 加上session的ID

  2. EL语言

    • 提供了一个测试对象是否为空格的特殊运算符empty 为空返回true( ! 非)

      ${empty var}

    • EL表达式中无法直接访问JSP文件中的隐含对象

    • 变量不是JSP文件中的局部变量或实例变量，而是放在特定范围内的属性。

    • 读取HTTP请求参数，形式为\${param[“var”]}或\${param.var}

  3. Filter要在web.xml中进行注册以及配置要过滤的内容，在doFilter对网页传入信息和配置信息中的内容比较，如果相同，则拒绝访问

  4. Velocity模板语言

    • 先创建一个.vm模板 ，然后再对Servlet中进行编写
    • loadConfiguration方法类似于Servlet的Init方法，区别在于返回的为Properties对象
    • handleRequest类似于Servelt的doGet和doPost方法，区别在于传入值中多了一个Context类型的参数，用与存放显示到HTML页面上的数据
    • 使用##单行注释，# # 多行注释
    • VTL是由一个$为前导，必须以一个字母开始
    • 给变量赋值一种方法是在java代码中给变量赋值，要么是通过#set指令给变量赋值#set($foo=”bar”)
    • .同其他语法 表属性。给属性赋值有两种方法，一种是创建Hashtable对象，把所有的属性保存在这个Hashtable中，再把这个对象保存在Context中。另外一种是定义一个JavaBean类
    • 正式引用符{}
    • 可以捕捉和九针代码中可能出现的大小写错误
    • if foreach等语法用#开始 用#end结束
    • 当使用#set指令时，在双引号中的字面字符串将被解析。包括在单引号中将不被解析
    • #include指令用来导入本地文件，超过一个文件，期间用逗号隔开
    • #parse用于导入并解析文件，只允许导入一个文件。
    • #marco指令 用于定义重复的段，使得模板更加简洁、便于维护#marco(name)+#end进行命名，可以带参数，调用时形式为#name()
    • 只有证书才可以进行数学运算，如果执行非整数的数学运算，则操作被记录到日志中，并返回Null
    • 字符串的连接只需要把这些字符串“放在一起”即可​
